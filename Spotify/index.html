Why Spotify?

The more I read about Spotify and working for Spotify, the more I know it's a company I want to work for. A love for music sparked the initial interest, as it does for most people, but after doing more research there's so much more beyond that. I've developed a few crushes on current and former Spotify employees like Tobias van Schnieder and Amber Grimes. These are the kind of people I want to be around. There's a quote that goes, "If you're the smartest person in the room, you're in the wrong room." I think Spotify is the right room for me. As great as Spotify is, I also know that there's a lot of room for improvement. I think Spotify would do well to emphasize more of its social aspects because that's where it differentiates itself from other music streaming services. I love that Spotify has a community forum where users can give kudos to ideas they think should be implemented, and I want to be a part of enhancing the Spotify experience for users, especially since I'm an avid user myself.

reaDefine

reaDefine is a Java program that incorporates the Merriam Webster Dictionary API to output a vocabulary list from an inputted text file. The program parses the text file to remove duplicate and common words, then calls on the API to define the remaining words, then uses this vocabulary list to generate a quiz for the user. The user can choose how many items are on the vocabulary list. Download the program here.
